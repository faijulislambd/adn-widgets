Executive summary
Multiple bank clients experienced delayed SMS delivery due to a partial connectivity loss between ADN's SMS platform and a third-party carrier. The issue has been fully resolved with permanent infrastructure improvements deployed.
Incident type
NetworkServer
Affected clients
BRAC Bank Dhaka Bank City Bank Eastern Bank Mutual Trust Bank
Root cause
Asymmetric network path between ADN SMS server and Infozillion caused path congestion and delivery queue buildup at the carrier end.
Affected system
ADN SMS Platform — GP single delivery path. DLR detection mechanism impacted. Queue overflow at third-party layer.

Immediate actions taken
Path configuration updated
Upload and download paths reconfigured from asymmetric to symmetric routing to eliminate path conflicts.
Enhanced monitoring deployed
Real-time path congestion monitoring with automated alerts activated across all carrier paths.
Path management logic enhanced
Additional validation checks added to the path management layer to prevent recurrence of this class of failure.
Redundancy & failover mechanisms added
Additional redundant paths and automatic failover capability implemented for uninterrupted service continuity.

Long-term remediation
1
Advanced path-shifting algorithms
Developing more sophisticated algorithms for detecting and preventing path-down issues, with automatic shifting to redundant and tertiary paths.
2
System audits and stress testing
Regular system audits and stress testing scheduled to identify and rectify any potential weaknesses in infrastructure before they manifest.
3
Comprehensive staff training
Ongoing training programs for technical staff to ensure they are equipped with the latest knowledge and skills to manage and troubleshoot such issues effectively.
Incident closed — all systems restored to normal operation 03 Aug 2024, 18:49 BST
Lessons learned
Cross-team communication and strict process compliance were the critical success factors in containing the impact. Long-term investment in automated path-shifting algorithms and regular infrastructure stress testing has been formally prioritized.
