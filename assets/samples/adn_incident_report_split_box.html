Executive summary
Multiple bank clients experienced delayed SMS delivery due to a partial connectivity loss between ADN's SMS platform and a third-party carrier. The issue has been fully resolved with permanent infrastructure improvements deployed.
Incident type
NetworkServer
Affected clients
BRAC BankDhaka BankCity BankEastern BankMutual Trust Bank
Root cause
Asymmetric network path between ADN SMS server and Infozillion caused path congestion and delivery queue buildup at the carrier end.
Affected system
ADN SMS Platform — GP single delivery path. DLR detection mechanism impacted. Queue overflow at third-party layer.
Immediate actions taken
Path configuration updated
Upload and download paths reconfigured from asymmetric to symmetric routing to eliminate path conflicts.
Enhanced monitoring deployed
Real-time path congestion monitoring with automated alerts activated across all carrier paths.
Path management logic enhanced
Additional validation checks added to the path management layer to prevent recurrence of this class of failure.
Redundancy & failover mechanisms added
Additional redundant paths and automatic failover capability implemented for uninterrupted service continuity.
